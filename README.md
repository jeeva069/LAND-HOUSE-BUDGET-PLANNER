# AI-Based Smart House Construction Budget Planning System

## Project Description

AI-Based Smart House Construction Budget Planning System is a full-stack
web application designed to help users estimate, manage, and optimize
house construction expenses before starting a project.

The system: - Calculates material, labor, engineering, and construction
costs - Compares spending with available budget - Provides smart
optimization suggestions - Displays budget status indicators - Shows
blueprint visualization examples
